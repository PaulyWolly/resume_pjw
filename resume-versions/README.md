# Resume versions (comparison)

Three layouts share the same data in `resume-app/src/app/data/resume.data.ts`.

| Version | Component | Description |
|--------|-----------|-------------|
| **Resume 1** | `components/resume1` | Current Angular / Netlify look (default) |
| **Resume 2** | `components/resume2` | JobLeads / Masterclass (Eleanor Smith–style) |
| **Resume 3** | `components/resume3` | Same content, clean document layout |

Switch versions in the toolbar on the site. Default remains **Resume 1**.

The file `02-jobleads-masterclass-style.html` is an optional static draft; the live comparison is in the Angular app.
